# lunar-birthday-calendar
农历生日日历生成器
可以通过配置的农历生日，生成日历，方便到入到其他日历软件中，如 google 日历等。

## 开发
```
cargo run
```
访问 http://localhost:9000 即可看到根据 birthdays.json 生成的日历文件。
